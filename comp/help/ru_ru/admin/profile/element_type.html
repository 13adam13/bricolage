Имя, описание (Name, Description)
Введите имя Типа Элемента и краткое описание.
Страничный (Page)
Эта отметка делает Тип Элемента страницей. Это означает, что если Текст включает более одного Страничного элемента, то для каждого из них будет создаваться отдельный HTML-файл.
Фиксированный (Fixed)
Определяет будет ли Тип Элемента включать фиксированный URL. Обычно это используется в «Обложках (Cover Pages)», которые ссылались бы на несколько других страниц и чьи URI не зависели бы, например, от даты создания такой Обложки.
Связанный Текст, Медиа (Related Story, Media)
Все Элементы могут иметь один текст и/или Медиа объект, связанные с ними. Данные отметки позволяют пользователям сделать это.
Тип (Type)
Тип может принимать значения либо Текст, либо Медиа, либо Элемент. Элемент типа Текст или Медиа является элементом высшего уровня. Все другие элементы имеют тип Элемент, которые могут быть добавлены к другим элементам в качестве субэлементов. Создавая новый Тип Элементов, вам необходимо выбрать тип на первой странице, так как другие опции зависят от этого свойства.

Если вы хотите удалить Тип Элемента, поставьте отметку «Удалить этот Профиль». По завершении нажмите клавишу Сохранить.
