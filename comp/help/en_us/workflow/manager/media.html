Search for existing media documents in Bricolage in order to pull them into the current workflow and check them out. Click Advanced Search or Simple Search to switch between the simple and advanced search interfaces. Your last search results will be displayed every time you return to this page in the same workflow, for as long as your current Bricolage session lasts.

Simple Search
The default search is very simple: just enter a title, URL or keyword and click Search. Enter % to search for all media documents. Use SQL LIKE wildcards such as % to generalize your search.
Advanced Search
This search allows you to be more specific about the media you're trying to find. Fill in any or all of the fields and click Search. To restart your search with a fresh page, click Clear Values.
Media Found
Click View next to the desired media to view its profile. If the media document is not checked out to another user or in a different workflow, select the Checkout checkbox and click the Checkout button at the bottom of the page to check out the media document and move it to your workspace. If you have already checked it out, click Edit to access its profile.
Create a New Media
If you can't find what you need in the existing Media Documents, why not create a new one? This link will take you to the New Media profile.
