This is where you add and edit element types. Element Types are basically groupings of Elements, where certain properties of all the Elements of that type are set.

Search
Searches Element Types by name. Do a blank search to list all Element Types or click a letter for an alphabetical list.
Add a New Element Type
Click this link to create a new Element Type. It brings you to the Element Type Profile page.
Existing Element Types
Click Edit to modify an Element Type's profile. Click Log to view its history of actions. To delete an Element Type, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.
