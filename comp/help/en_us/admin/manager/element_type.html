This is the Element Type Manager, where you create and edit Element Types. Element Types are instances of Element Type Sets, and are the building blocks of story and media documents. You should first read Bric::ElementAdmin in order to understand how to plan your element hierarchy.

Search
Searches Element Types by name. Do a blank search to list all Categories or click a letter for an alphabetical list. You can also select a type of element to list using the 'Pick a Type' menu.
Add a New Element Type
Click this link to create a new Element Type. It brings you to the Element Type Profile page.
Existing Element Types
Click Edit to modify an Element Type's profile. Click Log to view its event history. To delete an Element Type, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.
