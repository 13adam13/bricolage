This is where you add and edit Media Types, also known as MIME types.

Search

Add a New Media Type
Click this link to create a new Media Type. It brings you to the Media Type Profile page.
Existing Media Types
Click Edit to modify a Media Type's profile. Click Log to view its event history. To delete a Media Type, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.
