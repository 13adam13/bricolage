Here you can adjust settings and change the way certain information is displayed. Search for the Preference you want to edit, or do a blank search to view all Preferences. To change a preference, click Edit to get to the Preference Profile page, edit the preference, and click Save.

Bricolage Instance Name
Label used in window titles and the Welcome message on the login page.
Character Set
The character set to use when interacting with Bricolage. Bricolage will send content to your browser using the selected character set, and you should enter new content using the same set.
Date/Time Format
How the date and time are displayed.
Default Asset Sort
The default asset (story, media, and template) field to use for sorting on desks and in the various views of assets, such as Find Stories.
Default Search
The default search, Simple or Advanced, to use in Find Media, Find Stories, and Find Templates.
Filter by Site Context
When enabled, this preference causes the assets displayed on in workflows and in the Find interfaces to only be in the site selected in the Site Context select list at the top of the screen. Does not apply if Bricolage manages only one site.
Language
The language to use for the display of Bricolage, including help, labels, buttons, etc.
List Name Format
How people's names (such as those of users and contributors) are displayed in lists.
Name Format
How people's names are displayed outside of lists.
Permissions Widget
The style of widget used in administering group permissions, either Radio or Select.
Search Results / Page
The number of rows displayed per page in searches and on desks.
Show Alpha Search
When enabled, this preference displays an list of links for each letter of the US-ASCII alphabet as links to do fast searches in the search managers.
Time Zone
The time zone to use when displaying dates, both in the UI and when outputting dates in templates.
