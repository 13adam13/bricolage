Here you can tell Bricolage to send you messages, or Alerts, when certain actions are carried out. For example, if you have submitted a new Story to be published next Tuesday, you can have Bricolage send you an Alert as soon as it goes up to let you know that the action has been executed.

Search
Searches active alert types by name. Do a blank search to list all alert types.
Add a New Alert Type
Click this link to create a new alert type. It brings you to the Alert Type Profile page. Enter the requested information and then click Save at the bottom of the page.
Existing Alert Types
Click Edit to modify an Alert Type Profile. Click Log to view the alert type's event history. To delete an alert type, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.

Note to system administrators. You have to specify an SMTP server in bricolage.conf or else your users probably won't receive emails.
