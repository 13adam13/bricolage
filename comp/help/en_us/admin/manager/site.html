This is where you create and edit sites.

Search
Searches active sites by name. Do a blank search to list all active sites.
Add a New Site
Click this link to create a new site. It brings you to the Site Profile page. Enter the requested information and then click Save at the bottom of the page.
Existing Sites
Click Edit to modify a Site Profile. Click Log to view the site's history of actions. To delete a site, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.
