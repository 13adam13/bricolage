New Media

Use this tool to generate files for the Media that will be used on your site. The process is simple:
1. Create a new file.
2. Upload a photograph, graphic or other type of Media to the file.
3. Check it into a public Desk for approval.
4. Publish it (and relate it to a Story, if desired).

Create a Media file

Click New Media from the left navbar and enter the following vital information:
Title: The name of the Media file.
Media Type: Choose what kind of Story you are creating.
Source: ??description pending
Category: Where on the site the Media will be published.
Priority: Tells users how urgent the Media is.
Cover Date: Select the date and time that will be displayed with the Media (this is not necessarily the publish date, but what appears to viewers to be the publish date).
When done, click Create. This brings you to the Media Profile page.
