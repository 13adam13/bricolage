Name, Beschreibung
Geben Sie dan Namen des Elementtyps und eine kurze Beschreibung ein.
Seite
Wählen Sie diese Option an, um den Elementtypen als Seite zu definieren. Dies bedeutet, daß wenn ein Beitrag mehr als ein Seitenelement besitzt, für jede Seite eine separate HTML-Datei erzeugt wird.
Fester URL
Dies bestimmt, ob der Elementtyp einen festen URL bekommt. Dies wird generell auf "Deckblattseiten" verwendet, welche zu mehreren anderen Seiten verlinken und dessen URIs z.B. nicht von dem Datum abhängen sollten, an dem das Deckblatt erzeugt wurde.
Verwandte Beiträge, Medien
Alle Elemente können mit einem Beitrags- und/oder Medienobjekt verknüpft oder assoziiert werden. Anwählen dieser Option erlaubt es den Benutzer, dies zu tun.
Typ
Der Typ kann entweder Beitrag, Medium oder Element sein. Ein Element vom Typ Beitrag oder Medium ist ein Element ersten Ranges (Top-Level Element), welches anderen Elementen als Subelement angefügt werden kann. Wenn Sie ein Element erstellen, müssen Sie den Typ auf der ersten Seite angeben, da andere Optionen von dieser Eigenschaft abhängen.

Wenn Sie den Elementtypen löschen wollen, wählen Sie 'Dieses Profil löschen. Wenn Sie fertig sind, klicken Sie auf Speichern.
