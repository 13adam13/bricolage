Eigenschaften
Tragen Sie einen Präfix (Dr., Dipl-Ing., etc), Vor-, Mittel- und Nachnamen, sowie einen eventuellen Suffix (Jr., Esq., etc.). Diese sind optional, aber Sie sollten wenigstens den Namen eintragen.
Rollen
Bestehende Rollen für den Mitarbeiter werden angezeigt. Sie können auch einen neuen Rollennamen hinzufügen. Durch Klick auf 'Weiter' selektieren Sie die aktuelle Rolle. Nochmaliger Klick auf 'Weiter' bringt Sie zum vollen Mitarbeiterprofil.
