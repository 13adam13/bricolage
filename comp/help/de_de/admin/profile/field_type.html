Eigenschaften
Geben Sie eine kurze Beschreibung und die maximale Feldlänge (falls zutreffend) an. Wählen Sie aus, ob das Feld vorgegeben und/oder wiederholbar sein soll.
Anzeigenattribute
Die aufgeführten Attribute variieren je nach Feldtyp, werden aber einige der nachfolgenden enthalten:
Mehrfache Zulassen
Dies gilt nur für Auswahlfelder und gibt an, ob der Anwender mehrere Punkte der Liste anwählen kann, was z.B. dadurch erzielt wird, daß beim auswählen mehrerer Punkte die Umschalttaste gedrückt wird. select
Titel
Wie 'Bezeichnung' aber für Datumsfelder. (date)
Spalten
Anzahl der Spalten im Textbereich. (textarea)
Vorgegebener Wert
Der vorgegebene Wert für dieses Feld. (textfield, radio, pulldown, select, textarea)
Gruppenbezeichnung
Wie 'Bezeichnung' aber für Radiofelder. (radio)
Bezeichnung
Die Bezeichnung die der Elementadministrator ist der Liste der Felder sehen wird. (textfield, radio, checkbox, pulldown, select, textarea)
Maximale Größe
Die maximale Grö&zslig;e dieses Feldes. (textfield, textarea)
Optionen, Bezeichnung
Eine Liste von Zeilen bestehend aus Werten und Bezeichnern. Beachten Sie, daß 'Optionen, Bezeichnung' das Format jeder Zeile bestimmt. "Optionen" ist z.B. das "value" Attribut eines <option>-Elementes innerhalb einer <select>-Liste, wobei "Label" dann das ist, was dem Anwender dargestellt wird. (radio, pulldown, select)
Zeilen
Anzahl der Zeilen eines Textbereiches. (textarea)
Größe
Die Größe dieses Feldes. (textfield, select)

Wenn Sie dieses Feld vom Element löschen möchten, setzen Sie ein Hälchen in den 'Dieses Profil löschen' Kasten.

Wenn Sie fertig sind, klicken Sie Speichern, oder Abbrechen um Ihre Änderungen zu verwerfen.
