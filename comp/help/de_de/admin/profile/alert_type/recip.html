Sie können spezifizieren, daß Meldungen entweder an Benutzergruppen oder an einzelne Benutzer gesendet werden.

Empfangsgruppen
Bestimmt daß eine Meldung an eine oder mehrere Gruppen von Benutzern gesendet wird. Wählen sie die Gruppen aus der linksseitigen Liste, Verfügbare Gruppen und klicken Sie auf den Knopf "Zur Liste hinzufügen". Die Gruppe wird zur Liste der aktuellen Gruppen hinzugefügt. Um eine Gruppe aus der Liste der aktuellen Gruppen zu entfernen klicken Sie auf den Knopf "Aus Liste entfernen".
Empfangsbenutzer
Bestimmt daß eine Meldung an eine oder mehrere Benutzer gesendet wird. Wählen sie die einzelnen Benutzer aus der linksseitigen Liste, Verfügbare Benutzer und klicken Sie auf den Knopf "Zur Liste hinzufügen". Der Benutzer wird zur Liste der aktuellen Benutzer hinzugefügt. Um einen Benutzer aus der Liste der aktuellen Benutzer zu entfernen klicken Sie auf den Knopf "Aus Liste entfernen".

Wenn Sie fertig sind, klicken Sie auf Speichern, oder klicken Sie auf Abbrechen um die Änderungen zu verwerfen.
