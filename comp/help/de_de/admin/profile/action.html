Dieses Formular wird zum hinzufügen oder modifizieren von Aktionen verwendet, welches die Arten von Bearbeitung sind, die auf den Output angewendet werden wenn es zu einem Ziel veröffentlicht wird.

Eigenschaften

Typ
Der Typ von Aktion, der angewandt wird. Die am häfigsten verwendete ist das ‘Verschieben’ (Move), welches das eigentliche Kopieren oder Hochladen der Ausgabe auf einen Server durchführt.
Reihenfolge
Hiermit kann die Reihenfolge, in der Aktionen beim Veröffentlichungsprozeß durchgeführt werden, gesetzt werden.
