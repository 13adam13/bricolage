Ein Server definiert einen spezifischen Ort wo Output kopiert oder hochgeladen wird, wenn er an sein Ziel veröffentlicht wird.

Eigenschaften

Hostname
Der Hostname oder doie IP-Adresse des Servers. Wenn Ihre Verteilmethode die Angabe einer spezifischen Portnummer benötigt, tragen Sie sie hier mit ein.
OS
Das Betriebssystem unter dem der Server läuft. Dies bestimmt wie die Dateinamen auf dem Server erstellt werden sollten.
Dokumenten-Stammverzeichnis
Der Pfad auf dem Server zum Stammverzeichnis der Dokumente, wo der Output hinkopiert werden sollte.
Benutzername
Der Benutzername für den Server (für FTP und SFTP uploads).
Paßwort
Das zu obigem Benutzernamen passenden Paßwort.
Cookie
Ein HTTP-Cookie, der von einigen ‘Verschiebemethoden’ (z.B. WebDAV) genutzt werden kann, um sich beim Server zu authorisieren.
Aktiv
Der Server wird nur benutzt, wenn er als ‘aktiv’ gekennzeichnet ist.
