Dieses Formular dient zum editieren oder zum erstellen von Zielen, und bietet Informationen darüber, wie das Ziel zum veröffentlichen von Inhalt verwendet werden kann.

Im ersten Schritt der Erstellung eines neuen Ziels sind nur die ersten zwei Teile des Formulars sichtbar. Wenn diese ausgefüllt sind, klicken Sie auf ‘Weiter’ um zum Rest des Formulares zu gelangen. Das komplette Formular wird auch beim editieren eines existierenden Ziels angezeigt.

Eigenschaften

Name
Der Name des Ziels. Dieser muß einzigartig sein, wird aber nur verwendet, um verschiedene Ziele besser unterscheiden zu können.
Beschreibung
Eine genauere Beschreibung des Zwecks des Zieles.
Verschiebungsmethode
Die Methode, die zum kopieren der Daten an das Ziel verwendet wird. Der tatsächlich zur Verfügung stehende Umfang an Optionen hängt davon ab, wie Ihre Bricolage-Installation konfiguriert wurde, kann aber folgendes beinhalten:
Dateisystem
Eine einfache Kopie über das Dateisystem.
FTP
Upload auf einen Zielserver mittels FTP.
SFTP
Upload auf einen Ziel server mittels SFTP (secure FTP).
WebDAV
Upload auf einen Zielserver mittels HTTP mit WebDAV.
Ressourcen kopieren
Falls angewählt, werden die zu verteilenden Ressourcen zunächst in ein temporäres Verzeichnis umkopiert, bevor ‘Aktionen’ darauf angewandt werden.
Veröffentlichung
Falls angewählt (was üblicherweise der Fall ist) wir dieses Ziel für die Veröffentlichung fertigen Outputs verwendet.
Vorschau
Falls angewählt kann dieses Ziel zur Vorschau verwendet werden.

Ausgabekanäle

In diesem Abschnitt werden die Ausgabekanäle, die fü dieses Ziel verwendet werden, gesetzt.

Aktionen

Verwenden Sie diesen Teil des Formulars um Aktionen, welches die Arten von Bearbeitung sind, die auf den Output angewendet werden wenn es zu einem Ziel veröffentlicht wird, hinzuzufügen, editieren oder zu löschen.

Üblicherweise gibt es nur eine Aktion vom Typ ‘Verschieben’, die die eigentliche Kopier- oder Upload-Operation durchführt.

Server

Verwenden Sie diesen Teil des Formulars um Server, welche definieren wie Output bei Veröffentlichung an die Server geschickt wird, zu definieren. Wenn z.B. die FTP-Methode verwendet werden soll, muß hier ein Benutzername und Paßwort für den Server angegeben werden.
