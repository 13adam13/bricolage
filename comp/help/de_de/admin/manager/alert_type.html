Hier können Sie Bricolage anweisen, Ihnen Meldungen oder Benachrichtigungen zuzusenden, wenn bestimmte Aktionen durchgeführt werden. Haben Sie z.B. einen neuen Beitrag eingereicht, der am nächsten Dienstag veröffentlicht werden soll, kann Ihnen Bricolage eine Benachrichtigung zusenden sobald der Beitrag versandt wurde, um Sie so zu informieren daß die Aktion durchgeführt wurde.
