User Override

With this feature, a permitted administrator can log in as another user without having to enter a password. This is useful for changing the password of a user who has forgotten it as well as retrieving items from the Workspace of a user who is unavailable.
Search: Searches active users by name. You can also call up a list of users by letter or view the full list by clicking All.

Users Found: Scroll down to the desired user account and click Log In. You will be sent to that user's Workspace.
