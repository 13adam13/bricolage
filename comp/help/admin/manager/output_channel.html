Output Channels

Output Channels are the destinations for your published items. The default Output Channel is the World Wide Web, which in most cases will be the only channel used.
Search: Searches Output Channels by name. Do a blank search to list all Output Channels or click a letter for an alphabetical list.
Add a New Output Channel: Click this link to create a new Output Channel. It brings you to the Output Channel Profile page, where you will enter the channel's name and description.
Existing Output Channels: Click Edit to modify an Output Channel's profile. Click Log to view its history of actions. To delete a channel, check the Delete box, scroll to the bottom of the page and click the Delete Checked button.
