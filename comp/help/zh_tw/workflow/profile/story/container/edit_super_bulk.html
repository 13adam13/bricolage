使用超級大量編輯介面，一次編輯所有的欄位和元素的子元素．欄位和子元素 用等於(=)符號作區分，欄位的key名稱或子元素．欄位標籤隨後的是空行， 然後是文字欄位．欄位文字假設不包含空行．子欄位標籤存在自己的線上，但 Their(他們)的欄位和子元素不能直接編輯．

例如，編輯一個元素，包含"Header(檔頭)"和"Paragraph(段落)" 欄位和子元素"Insert(插入)"超級大量編輯語法能看到這些：

=header

This is a header

=paragraph

First paragraph.

=paragraph

Second paragraph.

=inset

=paragraph

Third paragraph.

重新安排元素內容，簡單移動元素和把符合的標籤放到你所需要的地方．例如，假如 上個範例你要移動檔頭到第一個段落之後，簡單編輯它，看起來像這樣：

=paragraph

First paragraph.

=header

This is a header

=paragraph

Second paragraph.

=inset

=paragraph

Third paragraph.

節省空間和讓編輯比較簡單，你能選擇"Default Element(基本元素)" 是不需要標籤的．例如，假如"paragraph(段落)"被選擇成預設元素， 之前的範例重新排列像這樣：

First paragraph.

=header

This is a header

Second paragraph.

=inset

Third paragraph.

最後，注意空欄位仍須包含空白行．例如，假如上個範例包含了一個" link_text"欄位裡沒有文字，link_text顯示像這樣： 

First paragraph.

=header

This is a header

=link_text



Second paragraph.

=inset

Third paragraph.

注意額外空行之後，=link_text標籤．這語法必須要讓Bricolage知道 欄位的存在，才能使用，但不必包含資料．
